Python - Move or copy files

Python automation

Often we may need to copy or move files to another directory for automatization of certain tasks. We can do this easily with a few lines of code, by using shutil.

The basic command to copy a file would be just shutil.copy(src, dst), similarly we have shutil.move(src, dst) if we want to move the file, where src is the file to copy and dst is the destination folder.

Let us consider the following, we need to copy all files from a single directory to a new one. We have our folder ‘main_folder’ which contains 2 files called file_1.docx and file_2.xlsx and we want to copy them to the folder called ‘secondary_folder’ 

import shutil
 
files = ['main_folder//file_1.docx', 'main_folder//file_2.xlsx']
									 
for f in files:
	shutil.copy(f, 'secondary_folder')

This is simple enough, but you have to specify the actual files you want to move, so now let's assume that we want to move all files within the main_folder to be copied but without having to type them. In order to achieve this we need an additional python module called os.

From our example above let's check the files that exist in the main_folder directory by using the os.listdir(folder_path), where folder_path is the path to the folder we want to check.

import os
 
files = os.listdir('main_folder')
print(files)

Now we just need to build the correct path to the files and that can be achieved with yet another function from the os module, which is path.join. 

import shutil
import os
 
files = os.listdir('main_folder')
 
for f in files:
    shutil.copy(os.path.join('main_folder', f), 'secondary_folder')

So, now we have achieved the same result without having to manually specify every file, which is very handy when we are dealing with thousands of files.

We have covered the basics now, so let us go a bit further and imagine that we haven't created the secondary_folder, prior to copy the folders so let us do that by using os.mkdir(dir), where dir is the name of the directory we want to create.

os.mkdir('secondary_folder')

This functionality can be useful when dealing with large network folders that usually take too long to refresh or simply to allow automated backups.

Full Code

import shutil
import os

files = os.listdir('main_folder')
os.mkdir('secondary_folder')
									 
for f in files:
	src = os.path.join('main_folder',f)
	shutil.copy(src, 'secondary_folder')
